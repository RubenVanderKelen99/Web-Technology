Opgave A:

  1. stel de font size in de html in op 62.5% (waarom?)
  2. stel het standaard lettertype in de body in op Arial, Verdana, sans serif / 16px (in rem!)
  3. zet alle paragrafen op 12px
  4. zet de h3's schuin
  5. zet de spans met class 'keyword' vet
  6. zet h2 in hoofdletters
  7. alle links binnen de div met id 'content' zijn bruin (kleur #b65216) en vet; haal de lijntjes eronder weg
